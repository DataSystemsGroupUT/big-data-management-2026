## This is the practice material for Big Data Management course 2026 @ University of Tartu

This repository contains weekly practice materials for the flipped classroom format (Docker setup, notebooks, and short tasks).

### Lecturers

- Kristo Raun
- Hasan Tanvir
- Gamal Elkoumy

### How to use this repository

Each week has its own folder (for example week_01__.../) with instructions in README.md.  
Typical flow:
- Before the on-site session: complete the practice and be ready for the Moodle mini-test.
- During the on-site session: group presentations, quiz (every 2nd session), discussion.

### Requirements (recommended)

- Docker Desktop installed and running
- Git (or download ZIP)
- Web browser :-)
- University HPC VM (optional, you need to ask this from teaching staff)

### Repository structure

- week_01__.../ Week 01 practice
- week_02__.../ Week 02 practice
- …

Note: large datasets are not committed to GitHub. Each week’s instructions explain how to obtain the required data.
